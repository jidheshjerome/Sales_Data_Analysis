# Sales_Data_Analysis
A data analyst project examining store sales performance from 2023–2025 across regions, products, salespeople, and customer types, with the goal of identifying what drives profitability and where the business can improve.

This project analyzes 1,486 sales records (16 columns) spanning three years to evaluate overall sales performance and uncover actionable opportunities for growth. The dataset includes unit price, quantity sold, region, store location, salesperson, customer type, payment method, returns, and delivery timing.

Data Cleaning (Python): Loaded and explored the dataset with pandas, corrected data types for date fields, removed rows with significant missing data, and engineered new fields for total sales and profit.
Analysis (SQL): Queried the cleaned dataset in MS SQL Server to answer nine core business questions covering regional performance, product rankings, salesperson profitability, customer segments, returns, payment methods, store performance, delivery times, and regional management.
Visualization (Power BI): Built an interactive dashboard to surface key trends at a glance.
